<h1 align="center">📡 Sampark</h1>

<p align="center">
  <b>An Offline Communication Application using Mesh Networking</b>
</p>

<p align="center">
  Communicate anytime, anywhere — even without an internet connection.
</p>

<hr>

<h2>📖 Overview</h2>

<p>
<b>Sampark</b> is an offline communication application that enables users to exchange messages
without internet connectivity. It leverages <b>mesh networking</b> concepts to establish
device-to-device communication, allowing nearby devices to connect directly and relay messages
across the network.
</p>

<p>
This makes communication possible during network outages, in remote locations, or in emergency
situations where traditional internet-based messaging services are unavailable.
</p>

<hr>

<h2>❗ Problem Statement</h2>

<p>
In many real-world scenarios such as:
</p>

<ul>
  <li>🌪️ Natural disasters</li>
  <li>🏔️ Remote or rural areas</li>
  <li>📵 Internet or cellular network failures</li>
  <li>🚨 Emergency situations</li>
</ul>

<p>
Traditional messaging applications become unusable because they depend on an active internet
connection or mobile network.
</p>

<p>
<b>Sampark</b> addresses this challenge by enabling offline device-to-device communication,
ensuring users can continue exchanging messages without relying on external network
infrastructure.
</p>

<hr>

<h2>💡 Solution</h2>

<p>
Sampark creates a local mesh network where nearby devices communicate directly and forward
messages between one another. Instead of requiring internet access, each connected device acts
as a relay, extending the communication range and ensuring messages reach their destination
through intermediate devices.
</p>
